# Costa Rica WildLife Tours - App Móvil

Aplicación móvil para expediciones de vida silvestre, herping y conservación en Costa Rica, construida con Apache Cordova e Ionic Appflow.

## 🚀 Características

- Diseño minimalista y elegante
- Galería de imágenes con lightbox
- Contacto directo por WhatsApp y teléfono
- Formulario de contacto por email
- Navegación por categorías de servicios
- Slider automático en página principal
- Soporte offline
- Integración con redes sociales

## 📋 Requisitos previos

- Node.js 18+
- Java JDK 11
- Android Studio (para compilar APK)
- Xcode (para iOS, solo Mac)
- Ionic CLI (`npm install -g @ionic/cli`)
